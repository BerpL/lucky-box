# Aplicación SPA con React

Aplicación web de tipo Single Page Application (SPA) desarrollada con React.

## Descripción

Este proyecto es una aplicación web moderna que utiliza React como biblioteca principal para construir una interfaz de usuario interactiva. La aplicación implementa navegación mediante React Router, permitiendo cambiar entre diferentes vistas sin recargar la página completa.

## Características

- Navegación mediante React Router
- Interfaz de usuario responsive
- Arquitectura de componentes reutilizables
- Optimización de rendimiento con Vite
- Diseño moderno y limpio

## Tecnologías

- React 18
- React Router DOM
- Vite
- CSS3

## Instalación

1. Instalar las dependencias:
```bash
npm install
```

2. Iniciar el servidor de desarrollo:
```bash
npm run dev
```

3. Abrir el navegador en `http://localhost:3000`

## Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run preview` - Previsualiza la construcción de producción

## Estructura del Proyecto

```
src/
  components/     # Componentes reutilizables
  pages/          # Páginas de la aplicación
  App.jsx         # Componente principal
  main.jsx        # Punto de entrada
  index.css       # Estilos globales
```

## Desarrollo

El proyecto utiliza Vite como herramienta de construcción, lo que proporciona:
- Hot Module Replacement (HMR)
- Compilación rápida
- Optimización automática
